I usually keep to myself as far as info about _myself_ goes, but since
you are interested enough to go here, here's some info about me, and what I do.

My name is, if you couldn't already guess, Oliver Webb. I am at the time of
writing (April 2024) 15 and a high school student. And have been programming
and learning about how computers worked for a few years now.

My main area of interest is Linux development, which I've been doing for well
over a year at this point. I work on my own scripts to automate things occasionally.
But lately I've been directing my effort towards the [toybox project](https://www.landley.net/toybox).
Here's a list of the most notable scripts I've made:

- [cheat](https://github.com/oliverkwebb/cheat): An offline way to view
[cheat.sh](https://cheat.sh) cheat-sheets
- [moontool](https://github.com/oliverkwebb/moontool): A collection of
fun tools adapted from older 80's tools to do stuff with moon phases.
- [ytmpv](https://github.com/oliverkwebb/ytmpv): A dmenu script that provides
quick accses to proxies when youtube is being ~~hostile~~ "unreliable" with
ad-blockers.
- [This website and it's content](https://github.com/oliverkwebb/oliverkwebb.github.io):
A basic SSG made in makefile web content in markdown. With other scripts like a RSS
feed generator.

I have a large number of snippets and dotfiles that I could link here but I never
bothered to upload, maybe someday I'll post them here if anyone shows any interest.

This is **not** where the bulk of my experience or effort has went to, that would
be to the [toybox project](https://www.landley.net/toybox). Which is over 200
unix command line tools in _under 500 kilobytes_ of binary space. If you don't
know about it look at it, it's cool. I've implemented several (fairly minor)
commands, done a bunch of cleanup work on things like the xz decompressor and
the bc implementation. Added a bunch of stuff to its vi. Among many, many other
things. Like UTF-8 support in a
[awk implementation](https://www.github.com/raygard) meant for toybox.

That's all for now, See my other stuff if you wanna know more
